❓ 常见问题

简介​

以下是关于 AgentGPT 最常见的问题列表。如果您有任何未解答的问题，请在 Discord 或 GitHub 上联系我们的管理或开发团队。

代理问题​

AgentGPT说它创建了一个文件/数据库/脚本，我在哪里可以找到它？
目前，AgentGPT无法以这种方式输出，但这是我们正在积极研究的内容。 请关注我们的路线图，以了解此功能可能何时可用。
AgentGPT实际上是否向100个人发送了电子邮件？
没有！我们目前不支持此功能，但这是我们要实现的功能之一。查看我们的路线图，以了解此功能可能何时可用。 当此功能可用时，我们将确保验证像“发送电子邮件”这样的操作是否真正是您想要执行的操作 🙂
我的代理程序总是运行出错！
由于 API 和基础设施成本，我们必须限制代理程序的运行时间 😢。

为了规避这个问题，您可以访问我们的设置文档，使用自己的 API 密钥在本地托管 AgentGPT。 或者，您可以订阅我们的专业计划以增加限制。

我的代理程序一开始表现很好，但后来停止了！我能从上次离开的地方继续吗？
目前，所有代理程序运行都是相互隔离的，因此这是不可能的。 这是我们未来要添加的功能之一，但同时，您可以使用它为您生成的信息创建另一个 AgentGPT 运行。 请关注我们的路线图，以了解此功能可能何时可用。
AgentGPT能使用 GPT-4 吗？
AgentGPT 目前为免费用户使用 GPT-3.5，PRO 用户则可以使用 GPT-4。如果您有 GPT-4 的 API 访问权限，则可以在设置中使用密钥并访问 GPT-4。
为什么输出总是被截断？
输出越长，我们生成它的成本就越高。 因此，我们对输出长度进行了限制，这可能会导致较长的消息被截断。 如果您提供自己的 API 密钥，可以通过增加令牌数量来增加设置菜单中的高级设置中的输出长度。

其他问题​

这与 ChatGPT 有什么区别？
ChatGPT 是一个很棒的工具，可以让您提出特定的问题并获得结果。它还会跟随对话，因此在收到响应后，您可以继续与其交谈，并且它将记住（在限制范围内）之前讨论过的内容。

另一方面，AgentGPT 是一个 AI 代理平台。您可以配置代理以实现广泛目标，它将自动思考和执行任务以实现该目标。

这与 AutoGPT 有什么区别？
AgentGPT 和 AutoGPT 都是涉及自主 AI 代理的项目。AutoGPT 是一个本地运行的工具，而 AgentGPT 是一个基于 Web 的平台。
AgentGPT 能做什么？
AgentGPT 可以做很多事情，但我们也正在努力赋予它更多功能。请访问我们的用例页面，了解人们目前如何使用 AgentGPT。
AgentGPT 是否可以访问互联网？
目前还没有，但这很快就会来！请关注我们的路线图。

请访问我们的设置文档以诊断任何问题。如果您有未记录的问题，请在GitHub 上提交问题。
